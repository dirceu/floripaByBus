# floripaByBus

HTML5 mobile app that searches routes and timetables of Florian√≥polis's public transportation.

This app was build with [PhoneGap](http://phonegap.com) and [Ionic](http://ionicframework.com). You can run it on an emulator using:

```shell
$ npm install -g cordova ionic ios-sim
$ ionic platform add ios
$ ionic emulate ios
```
